# プログラミング課題 実施報告書

## 1. 作品名
**PROGRAM QUIZ - Professional Peak Edition**

## 2. 開発のねらい
本プロジェクトは、単なるクイズアプリの枠を超え、「実務レベルのWebフロントエンド・バックエンド技術の統合」をテーマに開発しました。モダンなユーザー体験（UX）と、堅牢なセキュリティ、そして高い保守性を兼ね備えたフルスタックアプリケーションを目指しました。

## 3. 技術的な「すごい」ポイント（技術的独自性）

本プロジェクトで実装した、特筆すべき技術ポイントは以下の通りです。

### ① Web Audio API による動的サウンド生成
アセット（音声ファイル）を使用せず、ブラウザの **Web Audio API** を用いてオシレーターから直接音波を生成しています。
- **メリット**: ファイルロードが不要で遅延がゼロ。アプリの軽量化（数KBの削減）に寄与。
- **実装詳細**: 正解・不正解時に周波数とエンベロープ（音量変化）を制御し、心地よいUIフィードバックを実現。

### ② Proxyオブジェクトを活用した「リアクティブな状態管理」
Vue.js や React のような外部フレームワークを使わず、Vanilla JS の **Proxy** を用いて独自の状態管理システムを構築しました。
- **メリット**: `state.score = 100;` と代入するだけで、自動的に DOM が更新される仕組みを自作。
- **実装詳細**: データ（Model）と表示（View）を完全に分離し、コードの可読性と拡張性を劇的に向上させました。

### ③ 徹底した「不正対策」と「サーバーサイド検証」
ランキングシステムの信頼性を担保するため、複数の防壁を実装しました。
- **時間計測**: `Visibility API` を監視し、タブを切り替えた瞬間にタイマーが作動しなくなる不正を防止。
- **二重検証**: クライアント側のスコアを鵜呑みにせず、サーバー（PHP）側でも問題データと正解情報を突き合わせ、再計算を行っています。
- **セッション連携**: ゲーム開始時にサーバーへ通知し、回答時間があまりにも短すぎる（自動ツールの疑い）場合に登録を拒否する仕組みを導入。

### ④ PWA (Progressive Web App) とオフライン完全対応
電波の悪い環境でも学習を継続できるよう、高度なオフライン機能を実装しました。
- **Service Worker**: 高度なキャッシュ戦略により、二回目以降のアクセスは瞬時に完了。
- **オフラインキューイング**: オフライン中に送信したランキングスコアを `localStorage` に一時保存し、オンライン復帰時に自動で再送するバックグラウンド処理を自作しました。
- **拡張データベース設計**: ランキングデータに「難易度」だけでなく「ジャンル（カテゴリ）」を保持するよう設計し、多角的なデータ分析と表示を可能にしました。

### ⑤ アクセシビリティとUXへのこだわり
- **Haptic Feedback**: スマホ利用時に正解・不正解で異なる振動パターン（Vibration API）を提供。
- **スクリーンリーダー対応**: `aria-live` を用いたアナウンス機能を実装し、視覚障害のあるユーザーでも状況を把握可能。
- **キーボードナビゲーション**: 矢印キーでの回答操作に対応。

## 4. 設計思想（クリーンコード）
- **ES Modules**: クラスごとにファイルを分割し、疎結合な設計を維持。
- **Web Components**: 独自のカスタム要素（`<quiz-timer>` など）を定義し、UIの再利用性を高めました。
- **Internationalization (i18n)**: 辞書データによる日本語・英語の完全切り替えに対応。

## 6. スコア計算仕様
本アプリでは、単なる正解数だけでなく、回答速度と正確性の維持（コンボ）を評価する動的なスコアシステムを採用しています。

### 計算式
1問あたりの獲得スコアは以下の式で算出されます：
`獲得点数 = floor((基本点 + タイムボーナス + コンボボーナス) × 難易度倍率)`

- **基本点**: 100点
- **タイムボーナス**: `残り時間(秒) × 10`（最大100点）
- **コンボボーナス**: `現在の連続正解数 × 10`（2連続正解目から付与）
- **難易度倍率**:
    - Lv.1: 1.0倍
    - Lv.2: 1.2倍
    - Lv.3: 1.5倍
    - Lv.4: 2.0倍
    - Lv.5: 3.0倍

### 理論値（最大スコア）
全問正解かつ各問題を即答（残り時間10.0s）した場合、理論上の最大スコアは **7,620点** です（難易度 Lv.5 時）。

## 7. 自己評価
本アプリは、一見シンプルなクイズに見えますが、その裏側では最新のWeb標準APIを駆使し、フレームワークに頼らない「素のJavaScriptの力」を最大限に引き出しています。特に、セキュリティとオフライン対応の両立は、実務レベルでも通用する品質であると自負しています。
